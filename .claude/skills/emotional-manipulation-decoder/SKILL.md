# Emotional Manipulation Decoder

Skill pour décoder et comprendre l'ensemble des tactiques de manipulation
émotionnelle utilisées pour contrôler et influencer.

## Qu'est-ce que la Manipulation Émotionnelle?

### Définition

**Manipulation Émotionnelle**: Utilisation stratégique des émotions
(les vôtres ou les siennes) pour vous influencer, contrôler ou exploiter,
souvent de manière subtile et difficile à détecter.

### Le Triangle FOG

```
        FEAR (Peur)
            /\
           /  \
          /    \
         /      \
        /________\
OBLIGATION      GUILT
(Obligation)    (Culpabilité)

FOG = Brouillard qui obscurcit le jugement
Les 3 leviers émotionnels principaux
```

## Manipulation par la PEUR

### Types

```
1. PEUR DE L'ABANDON
   "Si tu fais ça, je te quitte"
   "Personne d'autre ne t'aimera"

2. PEUR DES CONSÉQUENCES
   "Tu vas le regretter"
   "Tu sais ce qui va arriver"

3. PEUR DE LA COLÈRE
   Explosions ou menaces d'explosion
   Marcher sur des œufs

4. PEUR DU JUGEMENT
   "Que vont penser les gens?"
   "Tu vas passer pour..."

5. PEUR DE LA PERTE
   "Tu perdras tout"
   Menaces sur les enfants, l'argent, la carrière
```

### Détection

| Signal | Exemple |
|--------|---------|
| Menaces voilées | "Je ne sais pas ce qui pourrait arriver..." |
| Ultimatums | "C'est ça ou..." |
| Prédictions négatives | "Tu vas échouer" |
| Rappel des punitions passées | "Tu te souviens la dernière fois?" |

## Manipulation par l'OBLIGATION

### Types

```
1. DETTE FORCÉE
   "Après tout ce que j'ai fait pour toi"
   "Tu me dois bien ça"

2. RÔLE IMPOSÉ
   "C'est ton devoir de [enfant/conjoint/actionnaire]"
   "Une bonne personne ferait..."

3. SACRIFICE INVOQUÉ
   "J'ai sacrifié X pour toi"
   "Je me suis privé pour que tu..."

4. NORME SOCIALE
   "C'est comme ça que ça se fait"
   "Tout le monde fait ça"

5. ENGAGEMENT PASSÉ
   "Tu avais promis"
   "On avait convenu que..."
```

### Détection

| Signal | Exemple |
|--------|---------|
| Rappels constants de "dettes" | "Je t'ai aidé en 2019..." |
| Définition de votre rôle | "Un vrai ami ferait..." |
| Invocation de sacrifices | "J'ai renoncé à X pour toi" |
| Pression de conformité | "Fais ce qu'il faut" |

## Manipulation par la CULPABILITÉ

### Types

```
1. CULPABILITÉ DIRECTE
   "Tu me fais du mal"
   "C'est de ta faute si je..."

2. MARTYR
   "Je souffre par ta faute"
   Soupirs, air triste, maladie opportune

3. COMPARAISON
   "[Autre personne] n'aurait jamais fait ça"
   "Au moins lui/elle..."

4. CULPABILITÉ PAR ASSOCIATION
   "Tu es égoïste/ingrat/méchant"
   Étiquettes négatives

5. CULPABILITÉ PRÉVENTIVE
   "Ne me déçois pas"
   "J'espère que tu ne vas pas..."
```

### Détection

| Signal | Exemple |
|--------|---------|
| "Tu me fais..." | "Tu me rends triste" |
| Position de victime | "Je souffre à cause de toi" |
| Labelling négatif | "Tu es tellement égoïste" |
| Attentes non dites puis déçues | "Je pensais que tu..." |

## Autres Tactiques Émotionnelles

### Love Bombing

```
TECHNIQUE:
Affection/attention excessive pour désarmer

SIGNAUX:
- Trop, trop vite
- Disproportionné
- Conditionnel (disparaît si résistance)
- Revient après conflit pour "réparer"
```

### Silent Treatment

```
TECHNIQUE:
Retrait punitif de communication/affection

SIGNAUX:
- Silence après désaccord
- Ignorer délibérément
- Punition sans explication
- Vous laisse anxieux/confus
```

### Gaslighting Émotionnel

```
TECHNIQUE:
Nier ou invalider vos émotions

SIGNAUX:
- "Tu es trop sensible"
- "Tu réagis excessivement"
- "Tu inventes des problèmes"
- "Ce n'était pas si grave"
```

### Triangulation Émotionnelle

```
TECHNIQUE:
Utiliser une tierce personne pour provoquer jalousie/insécurité

SIGNAUX:
- Mentionner des "ex" ou admirateurs
- Comparer à d'autres
- Créer de la compétition
```

### Victimisation

```
TECHNIQUE:
Se poser en victime pour éviter la responsabilité

SIGNAUX:
- Toujours quelque chose qui ne va pas
- Les autres sont toujours le problème
- Incapable de prendre en compte votre souffrance
```

## Répondre à la Manipulation Émotionnelle

### Principe Général

```
1. RECONNAÎTRE
   "C'est une tentative de manipulation"
   Nommer l'émotion: "Je ressens de la culpabilité"

2. PAUSE
   Ne pas réagir immédiatement
   L'urgence est souvent manufacturée

3. SÉPARER
   L'émotion ressentie ≠ obligation d'agir
   "Je me sens coupable ET je peux dire non"

4. RÉPONDRE (pas réagir)
   Depuis un lieu calme
   Focus sur les faits, pas les émotions
```

### Réponses par Tactique

| Tactique | Réponse |
|----------|---------|
| Culpabilisation | "Je comprends que tu ressens ça. Ma décision reste X." |
| Obligation | "Je ne suis pas d'accord sur cette dette." |
| Peur | "Les menaces ne m'aident pas à coopérer." |
| Love bombing | Méfiance, ralentir |
| Silent treatment | Pas de poursuite, vivre sa vie |
| Victimisation | "Je compatis ET mes limites restent." |
