# Codependency Pattern Detector

Skill pour identifier les patterns de codépendance qui rendent vulnérable
à l'exploitation narcissique.

## Qu'est-ce que la Codépendance?

### Définition

**Codépendance**: Pattern relationnel où l'estime de soi et l'identité
d'une personne dépendent excessivement de sa capacité à prendre soin
des autres et à être "nécessaire".

### Origine du Terme

```
Initialement: Partenaires d'alcooliques/addicts
Élargi à: Tout pattern de sur-investissement dans les autres
         au détriment de soi-même
```

## Caractéristiques de la Codépendance

### 1. Prise en Charge Excessive

```
MANIFESTATIONS:
- Faire pour les autres ce qu'ils devraient faire eux-mêmes
- Résoudre les problèmes des autres systématiquement
- Donner des conseils non sollicités
- Se sentir responsable du bien-être des autres

PENSÉES:
"Si je ne l'aide pas, qui le fera?"
"Il/elle a besoin de moi"
"Je suis le seul à pouvoir l'aider"
```

### 2. Faibles Limites

```
MANIFESTATIONS:
- Difficulté à dire non
- Accepte le manque de respect
- Laisse les autres décider pour soi
- Ne défend pas ses besoins

PENSÉES:
"Je ne veux pas créer de conflit"
"Ses besoins sont plus importants"
"Si je dis non, il/elle va partir"
```

### 3. Besoin de Contrôle (paradoxal)

```
MANIFESTATIONS:
- Tente de "réparer" les autres
- Donne pour contrôler la relation
- Anticipe les besoins pour éviter les crises
- Manipule pour que l'autre "aille mieux"

PENSÉES:
"Si je fais assez, il/elle va changer"
"Je sais ce qui est bon pour lui/elle"
```

### 4. Estime de Soi Externalisée

```
MANIFESTATIONS:
- Se valorise par ce qu'on fait pour les autres
- Besoin constant de validation
- Identité = être utile
- Panique si pas nécessaire

PENSÉES:
"Je suis quelqu'un de bien parce que je donne"
"Si je ne suis pas utile, je n'ai pas de valeur"
```

### 5. Négation de Soi

```
MANIFESTATIONS:
- Ignore ses propres besoins
- Ne sait pas ce qu'on veut vraiment
- Difficulté à identifier ses émotions
- Peu de temps/énergie pour soi

PENSÉES:
"Mes besoins ne sont pas importants"
"Je vais bien" (alors que non)
```

## Codépendance + Narcissisme: L'Alliance Toxique

### Pourquoi Ça "Matche"

```
CODÉPENDANT offre:              NARCISSIQUE cherche:
─────────────────────────────────────────────────────
Admiration et attention    →    Narcissistic supply
Tolère le mauvais traitement →  Quelqu'un qui reste
Prend soin sans limite     →    Être servi
Évite les conflits         →    Pas de confrontation
Excuse les comportements   →    Pas de responsabilité
Se blâme                   →    Quelqu'un à blâmer

= RELATION STABLE MAIS TOXIQUE
```

### Le Cycle Codépendant-Narcissique

```
1. N a un comportement abusif
        ↓
2. Codépendant excuse/pardonne
        ↓
3. N continue (pas de conséquence)
        ↓
4. Codépendant redouble d'efforts
        ↓
5. N prend plus
        ↓
6. Codépendant s'épuise mais reste
   (car partir = échec de "sauver")
```

## Sortir de la Codépendance

### Étapes de Guérison

```
1. RECONNAISSANCE
   "Je suis codépendant(e)"
   Ce n'est pas une vertu, c'est un pattern malsain

2. COMPRENDRE L'ORIGINE
   D'où vient ce pattern?
   Qu'est-ce qui m'a appris que ma valeur = être utile?

3. DÉVELOPPER L'IDENTITÉ
   Qui suis-je EN DEHORS des relations?
   Qu'est-ce que JE veux?

4. POSER DES LIMITES
   Apprendre à dire non
   Tolérer l'inconfort du non

5. LÂCHER LE CONTRÔLE
   Je ne peux pas "réparer" les autres
   Ce n'est pas ma responsabilité

6. AUTO-SOIN
   Mes besoins comptent
   Prendre soin de moi n'est pas égoïste
```

### Questions de Réflexion

```
"Si cette personne n'avait pas besoin de moi,
 est-ce que je resterais?"

"Est-ce que j'aime cette personne ou est-ce que
 j'ai besoin d'être nécessaire?"

"Qu'est-ce que JE veux, pas ce que je devrais vouloir?"

"Quand ai-je dit non la dernière fois?"

"Est-ce que je me sacrifie ou est-ce que je donne librement?"
```

### Différence: Don Sain vs Codépendance

| Don Sain | Codépendance |
|----------|--------------|
| Libre, sans attente | Attend réciprocité/reconnaissance |
| Respecte ses limites | Ignore ses limites |
| Peut dire non | Difficulté à refuser |
| Identité stable | Identité = être utile |
| Laisse l'autre responsable | Prend responsabilité de l'autre |
| Donne avec joie | Donne avec ressentiment caché |
