# Cognitive Dissonance Detector

Skill pour identifier et analyser la dissonance cognitive - l'inconfort mental
causé par des croyances, valeurs ou informations contradictoires.

## Qu'est-ce que la Dissonance Cognitive?

### Définition

**Dissonance Cognitive**: État de tension psychologique ressenti quand on
maintient simultanément des cognitions (croyances, attitudes, comportements)
contradictoires.

### Mécanisme

```
COGNITION A ←───── CONFLIT ─────→ COGNITION B
     │                                  │
     │         DISSONANCE               │
     │         (inconfort)              │
     └──────────────┬──────────────────┘
                    │
           BESOIN DE RÉSOLUTION
                    │
         ┌──────────┼──────────┐
         │          │          │
      Changer A  Changer B  Rationaliser
```

## La Dissonance dans l'Abus

### Exemple Classique

```
COGNITION A: "Cette personne m'aime"
     vs
COGNITION B: "Cette personne me maltraite"

RÉSOLUTION MALSAINE:
→ "Ce n'était pas vraiment de la maltraitance" (minimisation)
→ "Je l'ai provoqué" (auto-blame)
→ "Il m'aime tellement qu'il est jaloux" (rationalisation)
```

### Pourquoi C'est Problématique

```
La dissonance est INCONFORTABLE
     ↓
Le cerveau cherche à la RÉDUIRE
     ↓
Souvent en ADAPTANT la perception de l'abus
     ↓
Plutôt qu'en QUITTANT la relation
     ↓
Car quitter = admettre s'être trompé = autre dissonance
```

## Types de Dissonance dans la Manipulation

### 1. Dissonance Image/Réalité

```
"C'est quelqu'un de bien" vs "Il fait des choses terribles"

RÉSOLUTIONS TYPIQUES (malsaines):
- "Ce n'est pas vraiment lui"
- "C'est à cause de X (stress, alcool, moi)"
- "Au fond il est bon"
```

### 2. Dissonance Investissement/Résultat

```
"J'ai tout donné à cette relation/entreprise"
     vs
"La situation est mauvaise et ne s'améliore pas"

RÉSOLUTIONS TYPIQUES:
- "Ça va s'améliorer" (future faking accepté)
- "Je ne peux pas avoir gaspillé tout ça" (coût irrécupérable)
- "J'aurais dû faire plus"
```

### 3. Dissonance Identité/Situation

```
"Je suis une personne intelligente"
     vs
"Je suis dans une situation d'abus"

RÉSOLUTIONS TYPIQUES:
- "Ce n'est pas de l'abus" (protège l'image de soi)
- "N'importe qui pourrait être manipulé" (vrai mais utilisé pour rester)
```

### 4. Dissonance Paroles/Actions

```
"Il dit qu'il m'aime"
     vs
"Ses actions montrent le contraire"

RÉSOLUTIONS TYPIQUES:
- Croire les paroles, ignorer les actions
- "Il a du mal à exprimer l'amour autrement"
```

## Signaux de Dissonance

### Internes (ce qu'on ressent)

```
- Confusion mentale
- Anxiété inexpliquée
- Fatigue mentale
- Rationalisation excessive
- Justifications élaborées
- Évitement de certains sujets
- Inconfort quand quelqu'un pointe les contradictions
```

### Externes (ce qu'on observe)

```
- Défense illogique de l'abuseur
- Explications qui changent
- Irritation face aux questions
- Minimisation/Maximisation alternées
- "Oui mais..."
- Logique circulaire
```

## Réduire la Dissonance de Façon Saine

### Étapes

```
1. RECONNAÎTRE la dissonance
   "Je ressens un conflit entre X et Y"

2. NOMMER les cognitions contradictoires
   "Je crois A, mais je vis B"

3. ÉVALUER chaque cognition
   "A est-il basé sur des faits ou des souhaits?"
   "B est-il nié ou minimisé?"

4. CHOISIR la résolution saine
   - Modifier la CROYANCE si elle est fausse
   - Modifier l'ACTION si la croyance est vraie

5. TOLÉRER l'inconfort temporaire
   Changer est inconfortable mais nécessaire
```

### Questions pour Résoudre Sainement

```
"Si mon meilleur ami vivait ça, que lui dirais-je?"
"Est-ce que je minimise pour éviter de devoir agir?"
"Mes explications convaincraient-elles un observateur neutre?"
"Qu'est-ce que je sais être vrai au fond de moi?"
"De quoi ai-je peur si j'accepte la réalité?"
```

## Utilisation par les Manipulateurs

### Création Délibérée de Dissonance

```
TACTIQUE: Créer de la confusion pour paralyser

1. Affirmer une chose, faire l'opposé
2. Nier des réalités évidentes
3. Alterner affection et cruauté
4. Donner des informations contradictoires

→ Victime en dissonance constante
→ Énergie dépensée à "comprendre"
→ Moins d'énergie pour AGIR
```

### Exploiter la Dissonance Existante

```
"Tu sais que je t'aime" (après abus)
→ Offre une "résolution" facile
→ Permet de réduire la dissonance sans partir
→ Maintient la victime dans la relation
```
