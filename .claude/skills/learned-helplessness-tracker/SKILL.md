# Learned Helplessness Tracker

Skill pour identifier et adresser l'impuissance acquise - état psychologique
où une personne croit qu'elle ne peut pas influencer sa situation.

## Qu'est-ce que l'Impuissance Acquise?

### Définition

**Learned Helplessness** (Impuissance Acquise): État psychologique résultant
d'expériences répétées où les actions n'ont pas mené aux résultats souhaités,
conduisant à croire qu'on n'a aucun contrôle.

### Origine (Expérience de Seligman)

```
PHASE 1: Stimulus négatif inévitable
         (peu importe ce qu'on fait, même résultat)

PHASE 2: Apprentissage de l'impuissance
         (mes actions ne changent rien)

PHASE 3: Généralisation
         (même quand le contrôle est possible,
          on ne l'exerce plus)
```

## Développement dans l'Abus

### Comment ça se Développe

```
CYCLE DE CRÉATION:

1. TENTATIVE D'ACTION
   Victime: "J'ai essayé de lui parler calmement"

2. ÉCHEC/PUNITION
   N: Rage, punishment, situation empire

3. AUTRE TENTATIVE
   Victime: "J'ai essayé de lui donner de l'espace"

4. ÉCHEC/PUNITION (différent mais même résultat)
   N: Accusation d'abandon, manipulation

5. RÉPÉTITION
   Peu importe la stratégie → même résultat

6. CONCLUSION
   "Rien de ce que je fais ne change quoi que ce soit"

7. ARRÊT DES TENTATIVES
   Passivité, résignation, dépression
```

### Pourquoi les Manipulateurs la Cultivent

```
AVANTAGES POUR LE N:
- Victime plus facile à contrôler
- Moins de résistance
- Pas de tentatives de départ
- Pas de contestation des décisions
- Dépendance renforcée
```

## Manifestations

### Cognitives

```
PENSÉES TYPIQUES:
- "Ça ne sert à rien d'essayer"
- "Quoi que je fasse, ça ne changera rien"
- "Je suis piégé(e)"
- "C'est comme ça, il n'y a rien à faire"
- "Je n'ai pas le choix"
```

### Émotionnelles

```
RESSENTIS:
- Apathie
- Résignation
- Dépression
- Anxiété passive
- Engourdissement
- Désespoir
```

### Comportementales

```
COMPORTEMENTS:
- Passivité face aux décisions
- Absence d'initiative
- Non-réaction aux opportunités
- Difficultés à prendre des décisions
- Dépendance aux autres
- Procrastination
```

## Différence: Impuissance Acquise vs Réalité

### Question Critique

```
"Est-ce que je NE PEUX vraiment PAS agir,
ou est-ce que je CROIS ne pas pouvoir?"

VRAIE IMPUISSANCE:
- Contrainte physique réelle
- Absence totale de ressources
- Danger immédiat

IMPUISSANCE ACQUISE:
- Options existent mais non vues
- Ressources disponibles mais non sollicitées
- Généralisation d'échecs passés
```

## Surmonter l'Impuissance Acquise

### Étapes de Récupération

```
1. RECONNAÎTRE
   "Je suis peut-être en impuissance acquise"
   Ce n'est pas une faiblesse, c'est un apprentissage

2. IDENTIFIER L'ORIGINE
   Quelles expériences m'ont appris ça?
   Est-ce que le contexte a changé?

3. PETITES ACTIONS
   Commencer par des actions minuscules
   Restaurer le sentiment de contrôle progressivement

4. SUCCÈS PROGRESSIFS
   Chaque petit succès contredit l'impuissance
   Célébrer les petites victoires

5. SUPPORT EXTERNE
   Thérapie spécialisée
   Personnes qui croient en vous
   Modèles de sortie réussie

6. RECADRER
   "Cette stratégie n'a pas marché avec CETTE personne"
   ≠ "Rien ne marche jamais"
```

### Exercices Concrets

```
EXERCICE 1: Liste de contrôle
"Qu'est-ce que je PEUX contrôler aujourd'hui?"
- Mon réveil
- Ce que je mange
- Qui j'appelle
- Ce que je lis
...

EXERCICE 2: Preuve du contraire
"Quand ai-je réussi quelque chose?"
- Passé récent
- Passé lointain
- Petit ou grand

EXERCICE 3: Action minimale
"Quelle action minuscule puis-je faire maintenant?"
- Envoyer UN message à UN ami
- Faire UNE recherche sur les options
- Écrire UNE page de journal
```

### Questions de Recadrage

```
"Si mon meilleur ami était dans cette situation,
 que lui dirais-je sur ses options?"

"Y a-t-il UNE chose, même petite, que je pourrais faire?"

"Mes échecs passés prouvent-ils l'impossibilité future?"

"Quelqu'un d'autre dans une situation similaire a-t-il trouvé une issue?"
```
