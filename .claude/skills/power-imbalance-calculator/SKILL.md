# Power Imbalance Calculator

Skill pour calculer, quantifier et visualiser les déséquilibres de pouvoir
dans les relations personnelles et professionnelles.

## Pourquoi Mesurer le Pouvoir?

### Objectifs

```
1. CONSCIENCE - Voir clairement sa position
2. STRATÉGIE - Identifier où agir
3. PROTECTION - Anticiper les vulnérabilités
4. NÉGOCIATION - Connaître ses leviers
5. DÉCISION - Rester ou partir en connaissance de cause
```

## Les 7 Dimensions du Pouvoir

### 1. Pouvoir Économique/Financier

```
SOURCES:
- Revenu propre
- Patrimoine
- Accès aux ressources
- Dépendance financière
- Contrôle des comptes

CALCUL:
Qui peut survivre financièrement sans l'autre?
Qui contrôle les finances?
Qui a des alternatives économiques?
```

### 2. Pouvoir Informationnel

```
SOURCES:
- Accès à l'information
- Connaissances clés
- Secrets détenus
- Compétences uniques
- Réseau d'information

CALCUL:
Qui sait quoi?
Qui peut fonctionner sans l'autre?
Qui détient des informations sensibles?
```

### 3. Pouvoir Social/Relationnel

```
SOURCES:
- Réseau de support
- Alliances
- Réputation
- Connexions
- Influence sur les autres

CALCUL:
Qui a plus de soutien externe?
Qui peut mobiliser d'autres personnes?
Qui serait cru en cas de conflit?
```

### 4. Pouvoir Légal/Formel

```
SOURCES:
- Position hiérarchique
- Droits légaux
- Contrats favorables
- Pouvoir de signature
- Autorité formelle

CALCUL:
Qui a les droits?
Qui a l'autorité officielle?
Que disent les documents?
```

### 5. Pouvoir Physique/Sécurité

```
SOURCES:
- Force physique
- Contrôle du logement
- Capacité de nuire
- Sécurité personnelle

CALCUL:
Qui contrôle l'espace?
Y a-t-il menace physique?
Qui peut partir en sécurité?
```

### 6. Pouvoir Émotionnel/Psychologique

```
SOURCES:
- Stabilité émotionnelle
- Résistance à la manipulation
- Estime de soi
- Dépendance émotionnelle
- Attachement

CALCUL:
Qui est plus attaché?
Qui manipule qui?
Qui a besoin de qui émotionnellement?
```

### 7. Pouvoir d'Alternative (BATNA)

```
SOURCES:
- Options si la relation finit
- Capacité de repartir
- Alternatives viables
- Coût de sortie

CALCUL:
Qui a le plus à perdre en partant?
Qui a des alternatives?
Quel est le coût de sortie pour chacun?
```

## Grille de Calcul

### Évaluation par Dimension

Pour chaque dimension, évaluer de -5 à +5:
- **-5**: L'autre a tout le pouvoir
- **0**: Équilibre
- **+5**: Vous avez tout le pouvoir

```
| Dimension | Score -5 à +5 | Justification |
|-----------|---------------|---------------|
| Économique | | |
| Informationnel | | |
| Social | | |
| Légal/Formel | | |
| Physique/Sécurité | | |
| Émotionnel | | |
| Alternative (BATNA) | | |
| **TOTAL** | /35 | |
```

### Interprétation

```
+15 à +35: Vous avez significativement plus de pouvoir
+5 à +14: Léger avantage pour vous
-4 à +4: Relativement équilibré
-14 à -5: Désavantage pour vous
-35 à -15: Déséquilibre significatif contre vous
```

## Stratégies de Rééquilibrage

### Par Dimension

| Dimension | Comment Renforcer |
|-----------|-------------------|
| Économique | Revenu propre, épargne séparée, compétences |
| Informationnel | Accès à l'info, documentation, apprentissage |
| Social | Reconnecter avec proches, nouvelles alliances |
| Légal | Connaître ses droits, avocat, formaliser |
| Sécurité | Plan de sécurité, logement alternatif |
| Émotionnel | Thérapie, auto-suffisance, détachement |
| BATNA | Développer des alternatives, réduire le coût de sortie |

### Principes Généraux

```
1. DIVERSIFIER les sources de pouvoir
   Ne pas dépendre d'une seule dimension

2. RÉDUIRE les dépendances
   Chaque dépendance = vulnérabilité

3. DOCUMENTER
   L'information est pouvoir

4. CONSTRUIRE LENTEMENT
   Le rééquilibrage prend du temps

5. NE PAS ALERTER
   Le N résistera au rééquilibrage
   Discrétion recommandée
```

## Application au Contexte d'Actionnariat

### Exemple Spécifique

```
DIMENSION: Légal/Formel

| Facteur | Actionnaire A | Actionnaire B |
|---------|---------------|---------------|
| Parts | 23% | 48% |
| Voting power | Minoritaire | Majorité |
| Cautionnement | Oui, illimité | Oui, illimité |
| Position CA | Membre | Président |

→ Avantage net: Actionnaire B

IMPLICATIONS:
- A ne peut pas bloquer les décisions seul
- A porte risque disproportionné (23% parts, 50% risque)
- Besoin de coalition pour avoir du poids
```
