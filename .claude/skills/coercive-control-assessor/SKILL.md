# Coercive Control Assessor

Skill pour évaluer le niveau de contrôle coercitif exercé dans une relation
personnelle ou professionnelle.

## Qu'est-ce que le Contrôle Coercitif?

### Définition

**Contrôle Coercitif**: Pattern de comportements visant à dominer, isoler
et contrôler une personne, restreignant sa liberté et son autonomie.

C'est reconnu comme une forme d'abus dans plusieurs juridictions.

### Caractéristiques Clés

```
1. PATTERN - Pas un incident isolé, mais un système
2. INTENTIONNEL - Vise à contrôler
3. PROGRESSIF - S'intensifie avec le temps
4. INVISIBLE - Souvent non détecté de l'extérieur
5. DÉVASTATEUR - Impact profond sur la victime
```

## Les 8 Domaines du Contrôle Coercitif

### 1. Isolement

```
COMPORTEMENTS:
- Limiter contacts famille/amis
- Déménager loin des proches
- Critiquer l'entourage
- Monopoliser le temps
- Surveiller communications

PHRASES TYPES:
"Tes amis ne sont pas bons pour toi"
"Ta famille ne nous comprend pas"
"Tu n'as pas besoin d'eux, tu m'as moi"
```

### 2. Contrôle Économique

```
COMPORTEMENTS:
- Contrôler l'accès à l'argent
- Exiger justification des dépenses
- Empêcher de travailler
- Accumuler dettes au nom de l'autre
- Cacher l'état des finances

PHRASES TYPES:
"Je gère mieux l'argent"
"Pourquoi tu as besoin de travailler?"
"Tu n'as pas besoin de voir les comptes"
```

### 3. Surveillance

```
COMPORTEMENTS:
- Vérifier téléphone/emails
- Demander compte des déplacements
- GPS/tracker
- Appels/messages constants
- Arriver "par hasard"

PHRASES TYPES:
"Tu étais où exactement?"
"Pourquoi tu n'as pas répondu tout de suite?"
"Je veux juste savoir que tu vas bien"
```

### 4. Dégradation

```
COMPORTEMENTS:
- Critiques constantes
- Humiliations (public ou privé)
- Comparaisons défavorables
- Moqueries des émotions
- Attaques sur l'apparence/intelligence

PHRASES TYPES:
"Tu es tellement stupide"
"Personne d'autre ne voudrait de toi"
"Tu as de la chance de m'avoir"
```

### 5. Intimidation

```
COMPORTEMENTS:
- Regards menaçants
- Cris, objets cassés
- Menaces voilées ou explicites
- Démonstrations de force
- Punitions/représailles

PHRASES TYPES:
"Tu vas voir ce qui va arriver"
"Tu vas le regretter"
[Silence menaçant]
```

### 6. Contrôle des Activités

```
COMPORTEMENTS:
- Dicter l'emploi du temps
- Contrôler déplacements
- Interdire certaines activités
- Imposer des "règles"
- Punir les écarts

PHRASES TYPES:
"Tu n'as pas besoin de sortir"
"Je n'aime pas que tu fasses ça"
"Demande-moi d'abord"
```

### 7. Manipulation Émotionnelle

```
COMPORTEMENTS:
- Gaslighting
- Culpabilisation
- Silent treatment
- Love bombing cyclique
- Chantage émotionnel

PHRASES TYPES:
"Tu m'obliges à faire ça"
"Si tu m'aimais vraiment..."
"Après tout ce que je fais pour toi"
```

### 8. Contrôle via Enfants/Tiers

```
COMPORTEMENTS:
- Utiliser enfants comme messagers
- Menacer de prendre les enfants
- Monter les autres contre la victime
- Impliquer famille/amis dans le contrôle

PHRASES TYPES:
"Les enfants sauront que c'est ta faute"
"Ta famille pense que tu exagères"
```

## Grille d'Évaluation

### Scoring par Domaine

Pour chaque domaine, évaluer de 0 à 4:
- 0 = Absent
- 1 = Occasionnel/Léger
- 2 = Régulier/Modéré
- 3 = Fréquent/Sévère
- 4 = Constant/Extrême

```
| Domaine | Score 0-4 | Exemples |
|---------|-----------|----------|
| Isolement | | |
| Contrôle économique | | |
| Surveillance | | |
| Dégradation | | |
| Intimidation | | |
| Contrôle activités | | |
| Manipulation émotionnelle | | |
| Contrôle via tiers | | |
| **TOTAL** | /32 | |
```

### Interprétation

```
Score 0-8:   Relation potentiellement malsaine - vigilance
Score 9-16:  Contrôle coercitif probable - intervention recommandée
Score 17-24: Contrôle coercitif significatif - aide urgente
Score 25-32: Contrôle coercitif sévère - danger potentiel
```

## Différence: Contrôle vs Comportement Normal

| Comportement Normal | Contrôle Coercitif |
|---------------------|-------------------|
| S'inquiéter si pas de nouvelles | Exiger compte rendu constant |
| Discuter des finances ensemble | Contrôler tout l'argent |
| Exprimer une préférence | Imposer des règles |
| Conflit ponctuel | Pattern de dominance |
| Critique constructive | Dégradation systématique |
| Désaccord sur les amis | Isoler des proches |

## En Contexte Professionnel

### Contrôle Coercitif au Travail

```
MANIFESTATIONS:
- Micro-management extrême
- Surveillance excessive
- Isolation du reste de l'équipe
- Dénigrement constant
- Contrôle de la communication
- Menaces de licenciement
- Sabotage de carrière
```

### Spécificités Actionnariat

```
- Exclusion des décisions
- Opacité financière
- Dépendance économique créée
- Clauses contractuelles piégeantes
- Menaces sur l'investissement
```
