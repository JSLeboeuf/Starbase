## Structure Technique des Skills

Chaque skill suit le pattern Anthropic:

```

├── Frontmatter YAML
│   ├── name: identifiant unique
│   └── description: triggers d'activation
├── Définition/Introduction
├── Types/Catégories
├── Grille de Scoring
├── Template d'Analyse
├── Output Format
├── Contre-mesures/Recommandations
└── Ressources (references/, scripts/)
```
