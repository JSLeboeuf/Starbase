# Negotiation with Narcissist

Skill pour préparer et mener des négociations efficaces avec des personnes
présentant des traits narcissiques.

## Comprendre le Négociateur Narcissique

### Leurs Priorités (ordre d'importance)

```
1. LEUR IMAGE - Paraître gagnant
2. CONTRÔLE - Dominer la négociation
3. SUPPLY - Votre réaction émotionnelle
4. RÉSULTAT - L'accord lui-même (souvent secondaire)
```

### Leurs Tactiques Courantes

| Tactique | Description | Contre-mesure |
|----------|-------------|---------------|
| Intimidation | Volume, posture agressive | Rester calme, documenter |
| Fausses urgences | "C'est maintenant ou jamais" | Prendre son temps |
| Charme excessif | Love bombing pour désarmer | Rester factuel |
| Moving goalposts | Changer les termes | Tout par écrit |
| Victimisation | Se poser en victime | Ne pas culpabiliser |
| Menaces voilées | Sous-entendus menaçants | Clarifier, documenter |
| Dévalorisation | "Tu ne comprends rien" | Ne pas se justifier |
| Triangulation | Impliquer des tiers | Négocier en direct |

## Préparation (Critique)

### Avant Toute Négociation

```
PRÉPARATION - CHECKLIST
========================

1. OBJECTIFS CLAIRS
   - Idéal: Ce que je veux vraiment
   - Acceptable: Minimum que j'accepte
   - BATNA: Alternative si pas d'accord

2. DOCUMENTATION
   - Faits préparés
   - Preuves disponibles
   - Chronologie claire

3. ANTICIPATION
   - Ses arguments probables
   - Ses tactiques attendues
   - Mes réponses préparées

4. LOGISTIQUE
   - Lieu neutre si possible
   - Témoin présent
   - Durée limitée prévue
   - Sortie possible

5. ÉTAT D'ESPRIT
   - Objectifs clairs
   - Détachement émotionnel
   - Prêt à partir si nécessaire
```

## Pendant la Négociation

### Posture Générale

```
FAIRE:
✓ Rester calme et factuel
✓ Parler lentement
✓ Prendre des notes
✓ Demander des clarifications
✓ Reformuler pour confirmer
✓ Garder le contrôle du tempo

NE PAS FAIRE:
✗ S'énerver ou pleurer (supply)
✗ Se justifier excessivement
✗ Accepter sous pression
✗ Révéler sa stratégie
✗ Montrer du désespoir
✗ Engager sur des points émotionnels
```

### Techniques de Contrôle

#### 1. Le Disque Rayé

```
Répéter sa position calmement, sans variation:

N: "Tu es complètement déraisonnable!"
Vous: "Je comprends ton point de vue. Ma position est X."
N: "Après tout ce que j'ai fait!"
Vous: "Ma position reste X."
N: "Tu vas le regretter!"
Vous: "Je note ton opinion. Ma position est X."
```

#### 2. Le Brouillard

```
Absorber les attaques sans résister ni confirmer:

N: "Tu es égoïste!"
Vous: "Tu as peut-être raison." (neutre, pas d'engagement)
N: "Tu ne penses qu'à toi!"
Vous: "C'est possible." (continue sans s'arrêter)
```

#### 3. La Question Clarificatrice

```
Forcer la précision face aux vagues accusations:

N: "Tu sais très bien ce que tu as fait."
Vous: "Peux-tu être plus précis?"
N: "Tout le monde sait que tu..."
Vous: "Qui exactement? Quand?"
```

#### 4. Le Temps de Réflexion

```
Ne jamais accepter sous pression:

N: "J'ai besoin de ta réponse maintenant!"
Vous: "Je vais y réfléchir et te répondre d'ici [délai]."
N: "C'est maintenant ou jamais!"
Vous: "Si c'est ta position, je ne peux pas accepter
      sans réflexion. On peut se reparler [date]."
```

### Phrases Clés

**Poser des limites:**
> "Je suis prêt à discuter de X, pas de Y."
> "Ce n'est pas le sujet de notre discussion."
> "Revenons aux faits."

**Face aux attaques:**
> "J'entends que tu es frustré. Parlons des solutions."
> "Je ne suis pas ici pour débattre de mon caractère."
> "Concentrons-nous sur l'accord."

**Clarifier:**
> "Peux-tu préciser ce que tu veux dire?"
> "Concrètement, qu'est-ce que tu proposes?"
> "Mettons ça par écrit pour être sûrs."

**Terminer:**
> "J'ai besoin de temps pour réfléchir."
> "Nous n'avancerons pas aujourd'hui. Reprogrammons."
> "Je ne peux pas accepter ces conditions."

## Situations Spécifiques

### Négociation d'Actionnariat

```
PRÉPARATION SPÉCIALE:
- Convention d'actionnaires en main
- Calculs de valorisation
- Droits légaux clairs
- Avocat informé

POINTS CRITIQUES:
- Tout changement par écrit
- Témoins ou procès-verbal
- Délai de réflexion obligatoire
- Conseiller juridique avant signature
```

### Négociation de Sortie

```
OBJECTIFS CLAIRS:
- Valorisation juste
- Délais de paiement
- Libération des garanties
- Non-concurrence raisonnable

TACTIQUES ATTENDUES:
- Minimiser la valeur
- Maximiser les obligations
- Créer de l'urgence
- Culpabiliser

RÉPONSES:
- Évaluation indépendante
- Négociation point par point
- Délai de réflexion
- "Ce n'est pas personnel, c'est business"
```

### Négociation de Conflit

```
STRUCTURE:
1. Exposer les faits (pas les émotions)
2. Identifier les intérêts de chacun
3. Proposer des solutions
4. Négocier les détails
5. Formaliser l'accord

SI BLOCAGE:
- Médiateur externe
- Pause et reprise
- BATNA
```

## Après la Négociation

### Immédiatement Après

```
1. DOCUMENTER
   - Ce qui a été dit
   - Ce qui a été convenu
   - Ce qui reste en suspens

2. CONFIRMER PAR ÉCRIT
   - Email récapitulatif
   - "Pour confirmer notre discussion..."
   - Attendre confirmation écrite

3. ANALYSER
   - Qu'est-ce qui a fonctionné?
   - Qu'est-ce qui n'a pas fonctionné?
   - Ajustements pour la prochaine fois
```

### Email Type Post-Négociation

```

```

**Score > 20**: Négociation réussie
**Score 15-20**: Résultat acceptable
**Score 10-15**: Résultat mitigé
**Score < 10**: Revoir la stratégie

## Quand Ne Pas Négocier

```
ÉVITER LA NÉGOCIATION SI:
- État émotionnel fragile
- Pas de préparation possible
- Pas d'alternative (BATNA)
- Sécurité menacée
- Pression intenable

ALTERNATIVE:
- Reporter
- Déléguer (avocat, médiateur)
- Accepter le BATNA
- Refuser totalement
```
