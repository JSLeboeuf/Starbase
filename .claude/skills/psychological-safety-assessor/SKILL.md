# Psychological Safety Assessor

Skill pour évaluer le niveau de sécurité psychologique dans les environnements
relationnels, professionnels et personnels.

## Qu'est-ce que la Sécurité Psychologique?

### Définition

**Sécurité Psychologique**: Croyance partagée que l'environnement est sûr
pour la prise de risque interpersonnel - on peut être soi-même, poser des
questions, exprimer des préoccupations, admettre des erreurs sans craindre
punition ou humiliation.

### Composantes Clés

```
1. LIBERTÉ D'EXPRESSION
   Peut-on parler ouvertement?

2. ABSENCE DE PUNITION
   Y a-t-il des représailles pour les opinions?

3. RESPECT DES ERREURS
   Comment sont traitées les erreurs?

4. ACCEPTATION DE LA DIFFÉRENCE
   La diversité d'opinion est-elle tolérée?

5. PRÉVISIBILITÉ
   Les réactions sont-elles prévisibles et raisonnables?
```

## Indicateurs de Sécurité Psychologique

### Environnement SÛR

| Indicateur | Manifestation |
|------------|---------------|
| Questions bienvenues | On peut demander sans jugement |
| Erreurs = apprentissage | Pas de blâme, focus sur amélioration |
| Opinions diverses | Désaccord respectueux accepté |
| Feedback constructif | Critique sur les idées, pas les personnes |
| Prévisibilité | On sait à quoi s'attendre |
| Vulnérabilité possible | On peut admettre ne pas savoir |
| Conflits résolus | Désaccords traités sainement |

### Environnement DANGEREUX

| Indicateur | Manifestation |
|------------|---------------|
| Peur de parler | On marche sur des œufs |
| Blâme pour erreurs | Recherche de coupable |
| Pensée unique | Dissidence punie |
| Attaques personnelles | Critique vise la personne |
| Imprévisibilité | Réactions explosives possibles |
| Façade obligatoire | Montrer des faiblesses = danger |
| Conflits destructeurs | Explosions, rancune, représailles |

## Sécurité Psychologique par Contexte

### En Entreprise

```
SIGNES D'INSÉCURITÉ:
- Réunions où personne n'ose parler
- Idées tuées avant discussion
- Peur du manager
- Rumeurs/politique constante
- Turnover élevé
- Présentéisme (être là sans engagement)

SIGNES DE SÉCURITÉ:
- Débat constructif en réunion
- Erreurs discutées ouvertement
- Manager accessible
- Feedback bidirectionnel
- Rétention des talents
- Engagement authentique
```

### En Famille/Couple

```
SIGNES D'INSÉCURITÉ:
- Marcher sur des œufs
- Sujets tabous nombreux
- Peur des réactions
- Façade pour éviter conflits
- Secrets par peur
- Épuisement émotionnel

SIGNES DE SÉCURITÉ:
- Communication ouverte
- Désaccords gérés sainement
- Soutien dans la vulnérabilité
- Authenticité possible
- Respect mutuel constant
```

### En Actionnariat/Partenariat

```
SIGNES D'INSÉCURITÉ:
- Décisions sans consultation
- Information cachée
- Peur de poser des questions
- Représailles pour désaccord
- Atmosphère de méfiance
- Procès-verbaux édulcorés

SIGNES DE SÉCURITÉ:
- Transparence des informations
- Décisions collégiales
- Questions bienvenues
- Débat constructif
- Confiance mutuelle
- Documentation honnête
```

## Améliorer la Sécurité Psychologique

### Ce Qui Dépend de Vous

```
1. MODÉLISER
   Montrer sa propre vulnérabilité
   Admettre ses erreurs
   Poser des questions

2. RÉPONDRE POSITIVEMENT
   Remercier pour le feedback
   Ne pas punir les messagers
   Traiter les erreurs comme apprentissage

3. DEMANDER ACTIVEMENT
   Solliciter les opinions
   Créer l'espace pour parler
   Montrer que ça compte
```

### Ce Qui Dépend de l'Autre/du Système

```
Si la source d'insécurité est une personne:
- Confrontation (si sûr)
- Documentation
- Escalade (RH, avocat)
- Réduction d'exposition
- Sortie si nécessaire

On ne peut pas créer de sécurité
face à quelqu'un qui veut créer l'insécurité.
```
