# Exit Strategy Planner

Skill pour planifier une sortie sécurisée et stratégique des relations
professionnelles ou personnelles avec des personnalités narcissiques.

## Comprendre l'Exit

### Pourquoi une Stratégie?

Quitter une relation avec un narcissique sans préparation peut:
- Déclencher une rage narcissique
- Entraîner des représailles
- Causer des pertes financières
- Prolonger le conflit inutilement

**Une sortie planifiée minimise les dommages et maximise les chances de succès.**

### Types de Sortie

| Type | Description | Quand l'utiliser |
|------|-------------|------------------|
| **Clean Break** | Rupture totale, immédiate | Danger, pas d'obligations |
| **Graduelle** | Désengagement progressif | Obligations à démêler |
| **Négociée** | Accord formel de séparation | Business, actionnariat |
| **Légale** | Via avocat/tribunal | Conflit majeur, protection |

## Plan de Sortie par Contexte

### Exit d'Actionnariat

```
PHASE 1: PRÉPARATION (discrète)
────────────────────────────────
1. Documenter tout (preuves en lieu sûr)
2. Faire valoriser l'entreprise indépendamment
3. Consulter avocat en droit des affaires
4. Identifier les options dans la convention
5. Sécuriser copies de tous documents
6. NE PAS alerter le N encore

PHASE 2: POSITIONNEMENT
────────────────────────────────
1. Renforcer ses alliances si possible
2. Clarifier ses droits par écrit
3. Préparer sa proposition de sortie
4. Calculer son BATNA (pire cas acceptable)
5. Avoir son avocat prêt

PHASE 3: DÉCLENCHEMENT
────────────────────────────────
1. Annoncer l'intention (formel, par écrit)
2. Proposer les termes
3. Négocier (voir skill négociation)
4. Tout documenter

PHASE 4: EXÉCUTION
────────────────────────────────
1. Accord signé
2. Transfert selon l'accord
3. Libération des garanties
4. Clôture formelle
5. NE PAS revenir

PHASE 5: POST-EXIT
────────────────────────────────
1. Surveiller les obligations résiduelles
2. Maintenir la documentation
3. Grey rock si contact forcé
4. Reconstruire
```

### Exit Professionnel (Employé)

```
PHASE 1: PRÉPARATION
- Documenter toute forme de harcèlement
- Consulter avocat en droit du travail
- Chercher nouveau poste discrètement
- Préserver preuves de ses réalisations
- Copies de documents importants

PHASE 2: DÉPART
- Lettre de démission professionnelle
- Préavis légal minimum
- Transition minimale mais correcte
- Pas de confrontation inutile

PHASE 3: POST-DÉPART
- Références préparées (autres contacts)
- Prêt à répondre aux questions sur le départ
- Surveillance de diffamation possible
- Recours si représailles
```

### Exit Relationnel (Personnel)

```
PHASE 1: PRÉPARATION
- Sécurité d'abord (plan si danger)
- Finances séparées
- Documents personnels sécurisés
- Réseau de support alerté
- Logement prévu si nécessaire

PHASE 2: ANNONCE
- Lieu sûr ou public
- Court et définitif
- Pas de JADE (Justify, Argue, Defend, Explain)
- "Ma décision est prise"

PHASE 3: EXÉCUTION
- No Contact ou Grey Rock strict
- Changement d'habitudes
- Blocage sur tous les canaux
- Surveillance des hoovering

PHASE 4: PROTECTION
- Informer les proches
- Documentation si harcèlement
- Recours légaux si nécessaire
```

## Réactions du Narcissique à Anticiper

### Lors de l'Annonce

| Réaction | Probabilité | Réponse |
|----------|-------------|---------|
| **Rage** | Haute | Sortir, documenter, pas de supply |
| **Charme** | Haute | Méfiance, décision maintenue |
| **Victimisation** | Haute | Empathie limitée, pas de retour |
| **Menaces** | Moyenne | Documenter, avocat si sérieux |
| **Indifférence feinte** | Moyenne | Ne pas être dupe |

### Post-Exit (Hoovering)

```
FORMES DE HOOVERING:
- "J'ai changé" (faux)
- "Personne ne t'aimera comme moi"
- "Tu vas le regretter"
- "J'ai besoin de toi" (urgence fabriquée)
- Contact via tiers (flying monkeys)
- "Retrouvons-nous juste pour parler"

RÉPONSE:
- No Contact absolu si possible
- Grey Rock strict si contact forcé
- Pas d'engagement
- Documentation si harcèlement
```

## No Contact vs Grey Rock

### Quand Choisir

| Méthode | Quand l'utiliser |
|---------|------------------|
| **No Contact** | Pas d'obligation de contact, sécurité prioritaire |
| **Grey Rock** | Contact forcé (coparentalité, business, légal) |

### Implémentation No Contact

```
1. Bloquer sur tous les canaux
2. Informer les proches de ne pas transmettre
3. Ne pas répondre aux tentatives
4. Pas de surveillance des réseaux du N
5. Si contact: Ignorer ou documenter
6. Si harcèlement: Action légale
```

### Implémentation Grey Rock

```
1. Contact minimum nécessaire
2. Réponses courtes et factuelles
3. Pas d'émotion
4. Par écrit de préférence
5. Témoins si possible
6. Documentation systématique
```

## Reconstruction Post-Exit

### Priorités

```
1. SÉCURITÉ - Êtes-vous en sécurité?
2. STABILITÉ - Finances, logement, emploi
3. SUPPORT - Thérapie, proches, groupes
4. GUÉRISON - Temps, patience, travail sur soi
5. RECONSTRUCTION - Nouvelle vie sans le N
```

### Ce qui est Normal

- Douter de sa décision
- Tristesse même si bonne décision
- Anxiété face à l'inconnu
- Flashbacks
- Culpabilité injustifiée

### Red Flags de Retour

```
NE PAS RETOURNER SI:
- "Il/elle a changé" (sans preuves sur LONG terme)
- Sentiment de solitude
- Difficulté financière temporaire
- Pression des autres
- Nostalgie des bons moments
```
