## Utilisation Recommandée

### Flux d'Analyse Typique

```
1. DÉTECTION
   └── narcissistic-pattern-detector → Identification initiale
   └── [Skills tactiques spécifiques] → Comprendre les mécanismes

2. DOCUMENTATION
   └── evidence-compiler → Organiser les preuves
   └── timeline-builder → Chronologie des événements
   └── pattern-correlation-engine → Identifier les liens

3. ÉVALUATION
   └── legal-relevance-scorer → Valeur juridique
   └── witness-statement-analyzer → Témoignages

4. STRATÉGIE
   └── grey-rock-advisor → Protection quotidienne
   └── boundary-enforcement-guide → Maintenir les limites
   └── negotiation-with-narcissist → Si négociation nécessaire
   └── exit-strategy-planner → Planifier la sortie
```

```
└── Ressources (références, scripts)
```
