# Trauma Bonding Analyzer

Skill pour identifier et comprendre les mécanismes du trauma bonding -
l'attachement paradoxal qui se forme dans les relations abusives.

## Qu'est-ce que le Trauma Bonding?

### Définition

**Trauma Bonding** (Lien Traumatique): Attachement émotionnel fort et
paradoxal qui se développe entre une victime et son abuseur, résultant
de cycles répétés d'abus et de réconciliation.

### Mécanisme Neurobiologique

```
CYCLE HORMONAL:
1. ABUS → Cortisol + Adrénaline (stress)
2. RÉCONCILIATION → Dopamine + Ocytocine (soulagement/attachement)
3. CALME RELATIF → Anticipation anxieuse
4. RÉPÉTITION → Renforcement du lien

Le cerveau associe l'abuseur au SOULAGEMENT (pas à l'abus)
→ Attachement paradoxal
```

## Les 7 Étapes du Trauma Bonding

### 1. Love Bombing Initial

```
- Attention excessive
- Idéalisation de la victime
- Promesses grandioses
- "Âme sœur" instantanée
- Isolement progressif déguisé en exclusivité
```

### 2. Confiance et Dépendance

```
- Partage de vulnérabilités
- Création de dépendance (financière, émotionnelle, sociale)
- Distanciation des autres relations
- "Nous contre le monde"
```

### 3. Critique et Dévalorisation

```
- Critiques subtiles puis ouvertes
- Comparaisons défavorables
- Érosion de l'estime de soi
- "C'est pour ton bien"
```

### 4. Gaslighting et Confusion

```
- Réalité remise en question
- Doute de soi croissant
- Isolation de la validation externe
- "Tu es trop sensible"
```

### 5. Résignation

```
- Acceptation du traitement
- Adaptation au chaos
- Perte d'identité
- "C'est normal"
```

### 6. Perte de Soi

```
- Identité fusionnée avec l'abuseur
- Incapacité à imaginer la vie sans
- Défense de l'abuseur
- "Il a besoin de moi"
```

### 7. Addiction au Cycle

```
- Recherche du "high" de réconciliation
- Minimisation de l'abus
- Justification du comportement
- "Cette fois c'est différent"
```

## Le Cycle d'Abus

```
         ┌──────────────────────┐
         │   1. TENSION         │
         │   (accumulation)     │
         └──────────┬───────────┘
                    ↓
         ┌──────────────────────┐
         │   2. INCIDENT        │
         │   (explosion)        │
         └──────────┬───────────┘
                    ↓
         ┌──────────────────────┐
         │   3. RÉCONCILIATION  │
         │   (lune de miel)     │←── Dopamine/Ocytocine
         └──────────┬───────────┘    RENFORCEMENT
                    ↓
         ┌──────────────────────┐
         │   4. CALME           │
         │   (déni)             │
         └──────────┬───────────┘
                    │
                    └──────────→ Retour à 1.
```

## Indicateurs de Trauma Bonding

### Chez la Victime

| Indicateur | Manifestation |
|------------|---------------|
| Défense de l'abuseur | "Vous ne le connaissez pas vraiment" |
| Minimisation | "Ce n'était pas si grave" |
| Auto-blame | "J'aurais dû faire X" |
| Peur de partir | "Je ne survivrai pas sans lui" |
| Idéalisation | Focus sur les "bons moments" |
| Isolement | Éloignement des proches |
| Confusion | "Je ne sais plus ce qui est normal" |
| Retours répétés | Quitte puis revient |

## Différence avec l'Amour Sain

| Amour Sain | Trauma Bonding |
|------------|----------------|
| Sécurité constante | Alternance peur/soulagement |
| Respect mutuel | Dynamique de pouvoir |
| Communication ouverte | Marcher sur des œufs |
| Autonomie préservée | Dépendance créée |
| Support des proches | Isolement |
| Croissance mutuelle | Diminution de soi |
| Conflits résolus | Cycles répétitifs |

## Briser le Trauma Bond

### Étapes de Récupération

```
1. RECONNAISSANCE
   - Admettre la réalité de l'abus
   - Nommer le trauma bonding
   - Accepter que c'est un mécanisme, pas de l'amour

2. NO CONTACT / GREY ROCK
   - Couper le cycle de renforcement
   - Permettre le "sevrage" neurochimique
   - Protéger de la manipulation

3. SUPPORT PROFESSIONNEL
   - Thérapie spécialisée trauma
   - Groupes de soutien
   - Éducation sur les mécanismes

4. RECONSTRUCTION
   - Reconnecter avec soi-même
   - Reconstruire le réseau social
   - Établir de nouvelles bases
```

### Le "Sevrage"

```
SYMPTÔMES NORMAUX APRÈS SÉPARATION:
- Anxiété intense
- Pensées obsessionnelles
- Envie de contacter
- Idéalisation de l'abuseur
- Doute de sa décision
- Symptômes physiques

DURÉE TYPIQUE: 3-6 mois de symptômes aigus
(Comme un sevrage de substance)
```
