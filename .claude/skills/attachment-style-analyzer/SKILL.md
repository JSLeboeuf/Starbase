# Attachment Style Analyzer

Skill pour analyser les styles d'attachement et comprendre comment ils
influencent les dynamiques relationnelles et la vulnérabilité à la manipulation.

## Théorie de l'Attachement

---

### 1. Attachement Sécure (≈50% population)

**Caractéristiques:**
```
- À l'aise avec intimité ET autonomie
- Communication ouverte des besoins
- Gestion saine des conflits
- Confiance de base en soi et les autres
- Résilience émotionnelle
```

**En relation:**
```
- Partenaire fiable et présent
- Pas de jeux de pouvoir
- Capable de vulnérabilité
- Respecte les limites
```

**Vulnérabilité manipulation:** FAIBLE
- Reconnaît les red flags
- Quitte les situations toxiques
- Résiste à la culpabilisation

---

### 2. Attachement Anxieux-Préoccupé (≈20%)

**Caractéristiques:**
```
- Peur intense de l'abandon
- Besoin constant de réassurance
- Hypervigilance aux signaux relationnels
- Tendance à s'accrocher
- Faible estime de soi relationnelle
```

**En relation:**
```
- Cherche fusion/proximité
- Sensible aux fluctuations
- Peut devenir "collant"
- Donne beaucoup pour être aimé
```

**Vulnérabilité manipulation:** TRÈS HAUTE
- Love bombing = paradis
- Peur d'abandon exploitée
- Accepte mauvais traitement pour éviter rejet
- Hoovering très efficace

---

### 3. Attachement Évitant-Dédaigneux (≈25%)

**Caractéristiques:**
```
- Inconfort avec l'intimité
- Valorise l'indépendance extrême
- Minimise l'importance des relations
- Supprime/nie les besoins d'attachement
- Difficulté à exprimer émotions
```

**En relation:**
```
- Garde distance émotionnelle
- Se retire sous stress
- Peut paraître froid/distant
- Évite la dépendance
```

**Vulnérabilité manipulation:** MODÉRÉE
- Moins sensible au love bombing
- Mais peut ignorer les red flags par détachement
- Peut s'engager dans des relations superficielles toxiques

---

### 4. Attachement Anxieux-Évitant/Désorganisé (≈5%)

**Caractéristiques:**
```
- Désire ET craint l'intimité
- Comportement contradictoire (approche-évitement)
- Souvent lié à un trauma d'attachement
- Difficulté à réguler les émotions
- Relations chaotiques
```

**En relation:**
```
- Pattern push-pull
- Sabotage inconscient
- Réactions intenses et imprévisibles
- Difficulté à faire confiance
```

**Vulnérabilité manipulation:** TRÈS HAUTE
- Chaos interne exploité
- Cycle abusif peut sembler "normal"
- Difficulté à identifier ce qui est sain

---

## Dynamiques Dangereuses

### Combinaison Anxieux + Narcissique

```
POURQUOI ÇA "FONCTIONNE" (mal):

NARCISSIQUE offre:           ANXIEUX cherche:
- Attention intense initiale → Réassurance
- Intermittent reinforcement → Intensité émotionnelle
- Drama constant            → Engagement (même négatif)
- "Tu es spécial(e)"        → Validation

RÉSULTAT: Trauma bonding quasi-garanti
```

### Combinaison Évitant + Anxieux

```
DYNAMIQUE POURSUIVANT-DISTANT:

ANXIEUX: demande plus de proximité →
ÉVITANT: se retire →
ANXIEUX: anxiété augmente, demande plus →
ÉVITANT: se retire plus →

= Cycle auto-perpétuant de frustration
```

## Auto-Évaluation Rapide

### Questions Clés

**Pour l'Anxiété d'Attachement:**
```
1. Je m'inquiète souvent que mon partenaire ne m'aime pas vraiment
2. J'ai besoin de beaucoup de réassurance dans mes relations
3. Je m'accroche parfois quand je sens une distance
4. La peur d'être abandonné(e) me préoccupe
5. Je suis très sensible aux changements d'humeur de mon partenaire

Score: Nombre de "oui" × 2 = /10
```

**Pour l'Évitement d'Attachement:**
```
1. Je suis mal à l'aise quand les relations deviennent trop proches
2. Je valorise mon indépendance plus que l'intimité
3. Je me retire quand il y a trop d'émotions
4. J'ai du mal à dépendre des autres
5. Je trouve difficile d'exprimer mes émotions

Score: Nombre de "oui" × 2 = /10
```

### Interprétation

```
Anxiété 0-3, Évitement 0-3: Probablement SÉCURE
Anxiété 4-10, Évitement 0-3: Tendance ANXIEUSE
Anxiété 0-3, Évitement 4-10: Tendance ÉVITANTE
Anxiété 4-10, Évitement 4-10: Tendance DÉSORGANISÉE
```

## Vers un Attachement Plus Sécure

### Principes

```
1. CONSCIENCE - Identifier son style
2. ORIGINE - Comprendre d'où ça vient (souvent enfance)
3. PATTERNS - Reconnaître ses patterns automatiques
4. CHOIX - Faire des choix conscients différents
5. RELATIONS - Choisir des partenaires sécures
6. THÉRAPIE - Travail approfondi si nécessaire
```

### Pour les Anxieux

```
- Développer l'auto-validation
- Tolérer l'incertitude relationnelle
- Diversifier les sources de support
- Reconnaître le pattern de poursuite
- NE PAS choisir des évitants/narcissiques
```

### Pour les Évitants

```
- Reconnaître les besoins d'attachement (normaux)
- Pratiquer la vulnérabilité graduelle
- Rester présent sous stress
- Communiquer le besoin d'espace (au lieu de fuir)
- Choisir des partenaires patients mais pas anxieux
```
