## ETAPE 5 : RECOMMANDATION STRATEGIQUE

### L'ordre optimal

```
1. ESSAIE D'ABORD --> OPTION D (Compensation)
         |
         | Si refuse
         v
2. ENSUITE --> OPTION A (Defendre) + temps
         |
         | Si situation se degrade
         v
3. PUIS --> OPTION B (Vente negociee)
         |
         | Si refuse ou offre trop basse
         v
4. ENFIN --> OPTION C (Forcer rachat)
```

### Pourquoi cet ordre?

| Etape | Raison |
|-------|--------|
| D en premier | Peut te donner PLUS sans perdre tes parts |
| A en deuxieme | Position de force, temps de voir |
| B en troisieme | Sortie propre si D et A echouent |
| C en dernier | Option nucleaire, relations detruites |
