# ANALYSE STRATEGIQUE - LOYAUTE JEROME
## Risque de depart et alternative : employe loyal sans equite

---

## LE PROBLEME STRATEGIQUE

### Ce que vous observez :
> "Des qu'il va avoir un peu d'argent, il va faire ses propres projets lui-meme quand on pourrait avoir un employe loyal qui construit des cliniques sans equite"

**ANALYSE** :
- Jerome n'est **pas loyal** a long terme
- Il va **partir** des qu'il aura les moyens
- Vous perdez l'equite que vous lui donnez
- Il y a une **alternative meilleure** : employe loyal sans equite

---

## RISQUE DE DEPART DE JEROME

### Scenario probable :

**1. Court terme (maintenant)** :
- Jerome demande plus de parts (6% - 23%)
- Il veut "justifier" sa contribution
- Il se positionne pour partir

**2. Moyen terme (6-12 mois)** :
- Il accumule de l'argent (6% sur vos projets)
- Il apprend le metier (financement, construction, developpement)
- Il construit son reseau

**3. Long terme (12-24 mois)** :
- Il a assez d'argent pour partir
- Il a assez d'experience pour etre independant
- **Il part et fait ses propres projets**
- Vous perdez l'equite que vous lui avez donnee

## ALTERNATIVE : EMPLOYE LOYAL SANS EQUITE

### Option 1 : Charge de projet salarie

**Structure** :
- **Salaire** : 120,000$ - 150,000$/an
- **Bonus** : 10,000$ - 20,000$ par projet reussi
- **Equite** : **0%** (aucune)

**Avantages** :
- **Pas de perte d'equite** si l'employe part
- **Loyaute** : l'employe a interet a rester (salaire stable)
- **Controle** : vous gardez le controle total
- **Pas de conflit** : pas de negociation de parts
- **Remplacable** : si l'employe part, vous en engagez un autre

**Cout** :
- 120,000$ - 150,000$/an (vs 6% d'equite qui peut valoir beaucoup plus)

### Option 2 : Employe avec bonus performance

**Structure** :
- **Salaire de base** : 80,000$ - 100,000$/an
- **Bonus par projet** : 15,000$ - 30,000$ selon performance
- **Equite** : **0%** (aucune)

**Avantages** :
- **Motivation** : bonus lie a la performance
- **Pas de perte d'equite** si l'employe part
- **Flexibilite** : vous ajustez les bonus selon les resultats
- **Pas de conflit** : pas de negociation de parts

## CALCUL FINANCIER

### Scenario 1 : Garder Jerome avec 6% equite

**Cout sur 5 ans** :
- 6% sur 450 Larocque (1,440,000$) = **86,400$**
- 6% sur 6PLEX (~2,500,000$) = **150,000$**
- 6% sur Dick Ledoux (3,000,000$) = **180,000$**

- **Total** : **716,400$** en equite

**Risque** :
- Jerome part dans 2 ans
- Vous perdez **716,400$** en equite
- Il devient un concurrent

### Scenario 2 : Engager un employe loyal

**Cout sur 5 ans** :
- Salaire 130,000$/an x 5 ans = **650,000$**
- Bonus (25,000$/an x 5 ans) = **125,000$**
- **Total** : **775,000$** en salaire

**Avantages** :
- Pas de perte d'equite
- Employe loyal (reste)
- Controle total
- Pas de conflit

**Difference** :
- **775,000$** (salaire) vs **716,400$** (equite)
- Cout similaire, mais **pas de perte** si l'employe part
- Et **pas de concurrent** si l'employe part
