## LA STRUCTURE DE FINANCEMENT

### QUI FINANCE A 100% ?

**Structure actuelle** :
- **Cedric** : Apporte le capital (financement)
- **Cedric** : A 48% des parts
- **Cedric** : Finance a 100% certains batiments

### QU'EST-CE QUI SE PASSE POUR LES BATIMENTS FINANCES A 100% PAR CEDRIC ?

**Question cle** : Si Cedric finance a 100%, pourquoi n'a-t-il pas 100% de ces batiments ?

**Reponse** :
1. **Echange de services** : Vous/Felix/Jerome apportez du travail/expertise
2. **Deal sourcing** : Vous trouvez les opportunites
3. **Gestion** : Vous gerez les projets
4. **Sweat equity** : Vous echangez du travail contre des parts

**Structure typique** :
- **Cedric finance a 100%** = Il prend le risque financier
- **Mais il a besoin** de vous pour trouver/gerer les projets
- **Donc il partage** les parts avec vous
