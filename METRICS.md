## Core Velocity Metrics

### 1. Decision Velocity
> "Faster yeses and faster nos"

- **Target:** Décision en < 7 jours
- **Mesure:** Nombre de décisions qui dépassent 7 jours
- **Goal:** 0 décisions en attente > 7 jours

### 2. Ship Velocity
> "What have you accomplished this week?"

- **Target:** Min 3 livrables/semaine
- **Mesure:** Choses LIVRÉES (pas "en cours")
- **Goal:** Augmenter de 10% chaque mois

### 3. Blocker Resolution Time
> "Barriers are operational risks"

- **Target:** Blocker identifié → résolu en < 48h
- **Mesure:** Temps moyen de résolution
- **Goal:** < 48h average

### 4. Learning Capture Rate
> "We never fail = we never learn"

- **Target:** 100% des échecs documentés
- **Mesure:** Ratio échecs/learnings documentés
- **Goal:** 1:1 minimum

### 5. AI Integration Score
> "AI first across all domains"

- **Target:** 80%+ des tâches assistées par IA
- **Mesure:** Tâches avec IA / tâches totales
- **Goal:** Augmenter chaque mois

## Speed Score Calculator

**Calcule ton score hebdomadaire :**

```
Speed Score = (Ships × 2) + Decisions + Blockers_Killed + (AI_Usage/20)
```

| Score | Interprétation |
|-------|----------------|
| 15+ | Wartime speed |
| 10-14 | Acceptable, push more |
| 5-9 | Peacetime — accelerate like hell |
| <5 | Code red — major blockers à identifier |

## Anti-Metrics (ce qu'on ne mesure PAS)

- Heures travaillées ❌
- Réunions assistées ❌
- Emails envoyés ❌
- "Busy-ness" ❌

**On mesure les OUTPUTS, pas les INPUTS.**
